Pair Programming

                

                    
Humans are visual thinkers.

                    
Women more than men.

                    
Solving problems together allows you to separate the problem from the language.

                    
It also helps spot errors faster.

                    
                

            


            

                
Pair Programming - 101

                

                    
                    
    
                        
  • One person should "drive" - type the code, move the mouse, etc.
    • 
                        
  • The other should tell the first person how they think the problem should be solved.
    •
